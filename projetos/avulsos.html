Projetos Avulsos

Escrito por Mateus Galdino, em 02 de Maio de 2021.

Eu chamo de "projetos avulsos" aqueles projetos que desenvolvi enquanto aprendia uma linguagem de programação. Sendo, normalmente, projetos curtos com ideias curtas em mente. Eles podem ser voltados a uma funcionalidade da linguagem, como uma biblioteca, ou um conceito, como por exemplo a OO(Orientação a Objetos).

Eles estão divididos entre linguagem no meu Github, alguns em Javascript, voltados para o Front-end ou para o Backend com Node.js, outros em Java, voltados principalmente para a sua parte backend.

Avulsos 1 - Randomizer

Escrito por Mateus Galdino, em 02 de Maio de 2021.

Randomizer, ou aleatorizador ou randomizador (em tradução literal), é o primeiro projeto mais integrado que fiz dentro de java. Ele utiliza conceitos do fim do chamado "core-java"(o núcleo de aprendizado da linguagem Java), como os objetos, as Interfaces e algumas bibliotecas do SDK, principalmente a java.util.

Ele faz uso do método java.util.Random para criar números aleatórios e utiliza esses números em seus próprios métodos. Atualmente, o objeto Randomizer possui três métodos úteis:

  • O randomNumber, que utiliza o Random e produz um número dentro de um intervalo dado pelo usuário;
  • O randomArray, que produz um array de números dentro de um intervalo dado pelo usuário;
  • o corHex, que produz um código de cor hexadecimal aleatório para o usuário.

O randomizer já vem com um código de teste onde é apresentado as funcionalidades de cada código e como ele pode ser utilizado dentro de outro projeto. Se for de interesse, o código pode ser utilizado de forma prática, simplesmente ao ser baixado e chamada por randomizer.Randomizer.


  import randomizer.Randomizer; //importa o randomizer

  public class Teste {
                    
    public static void main(String[] args) {
        Randomizer r = new Randomizer(); //chama ele como objeto
        System.out.println(r.corHex()); //usa ele para printar uma cor
    }
  }
  //no meu teste, printou #6860EA
