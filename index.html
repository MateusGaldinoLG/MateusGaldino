Avulsos 2 - Lorem Ipsum Generator

Escrito por Mateus Galdino, em 22 de Junho de 2021.

O Lorem Ipsum Generator, ou gerador de Lorem Ipsum, é um projeto bem simples feito em Java. Ele é menor em funcionalidade que o Randomizer, mas ao mesmo tempo utiliza mais conceitos de pacotes de Java e até de desenvolvimento

Ele pode ser exportado a partir do module-info.java da aplicação que quer utilizá-lo. Ao ser instanciado como new LoremIpsumGenerator();, ele pode ser utilizado a partir de três métodos, dependendo de como o usuário queira gerar o Lorem Ipsum.

Seus métodos são:

  • generate(int numPalavras): que gera um lorem com uma quantidade de palavras dada pelo parâmetro
  • generate(int numPalavras, int numParagraph): que gera um lorem em x paragrafos a partir de y palavras totais
  • generateParagraphs(int numPalavras, int palavrasParagraph): que gera uma quantidade de parágrafos, cada um com x palavras, a partir de y palavras totais

Para gerar o Lorem, é utilizado um arquivo .txt dentro do projeto, que é lido pelo File da java.io e retornado na forma de String para o método principal.

Vai um exemplo de utilização: 


              module example {
                //...
                requires com.generator;
              }


              import com.generator.lorem.LoremIpsumGenerator;

              public class LoremExemplo {
                public static void main(String[] args) {
                  int tamanho = 10;
                  
                  LoremIpsumGenerator loremIpsumGenerator = new LoremIpsumGenerator();
                  
                  String lorem = loremIpsumGenerator.generate(tamanho);
                  System.out.println(lorem);

                }
              }

Avulsos 1 - Randomizer

Escrito por Mateus Galdino, em 02 de Maio de 2021.

Randomizer, ou aleatorizador ou randomizador (em tradução literal), é o primeiro projeto mais integrado que fiz dentro de java. Ele utiliza conceitos do fim do chamado "core-java"(o núcleo de aprendizado da linguagem Java), como os objetos, as Interfaces e algumas bibliotecas do SDK, principalmente a java.util.

Ele faz uso do método java.util.Random para criar números aleatórios e utiliza esses números em seus próprios métodos. Atualmente, o objeto Randomizer possui três métodos úteis:

  • O randomNumber, que utiliza o Random e produz um número dentro de um intervalo dado pelo usuário;
  • O randomArray, que produz um array de números dentro de um intervalo dado pelo usuário;
  • o corHex, que produz um código de cor hexadecimal aleatório para o usuário.

O randomizer já vem com um código de teste onde é apresentado as funcionalidades de cada código e como ele pode ser utilizado dentro de outro projeto. Se for de interesse, o código pode ser utilizado de forma prática, simplesmente ao ser baixado e chamada por randomizer.Randomizer.


  import randomizer.Randomizer; //importa o randomizer

  public class Teste {
                    
    public static void main(String[] args) {
        Randomizer r = new Randomizer(); //chama ele como objeto
        System.out.println(r.corHex()); //usa ele para printar uma cor
    }
  }
  //no meu teste, printou #6860EA

O que é o Dungeon Bot?

Escrito por Mateus Galdino, em 30 de Abril de 2021.

O Dungeon Bot é um bot voltado para o jogo Dungeons and Dragons 5e, usado na plataforma Discord

O jogo Dungeons and Dragons 5e, ou simplesmente D&D, possui um conjunto de regras bem robusto, indo desde classes até raças e funções próprias de armas e encontros. Sendo um Jogo de Role-playing (RPG), ele ganhou grande espaço em plataformas de conversas online, principalmente no periodo durante e após a pandemia. Nesse contexto, eu fui introduzido ao jogo e amei a primeira vista, ou ao primeiro jogo.

Minha mesa já mostrava a necessidade de conseguir acessar os conteúdos de D&D de forma rápida durante o jogo, sem precisar acessar as 300 páginas do livro do jogador(LDJ ou PHB), então, ao começar a programar, decidi começar um projeto de um bot para Discord, usando node.js e a biblioteca Discord.js, para compactar um conjunto de regras e recomendações em um único lugar mais acessível.
